## How to run this Application

```sh
$ git clone git@github.com:hayo0914/ReactNative-ImageBoardApp-Sample.git
$ cd ReactNative-ImageBoardApp-Sample

$ yarn install 
# or
$ npm install

# run on android
$ npm run android
# or 
$ yarn android

# run on ios
$ yarn ios
# or
$ yarn android
```
