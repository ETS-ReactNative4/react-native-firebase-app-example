## How to run this Application

```sh
$ git clone git@github.com:hayo0914/ImageBoardApp.git

$ yarn install 
# or
$ npm install

# run on android
$ npm run android
# or 
$ yarn android

# run on ios
$ yarn ios
# or
$ yarn android
```
